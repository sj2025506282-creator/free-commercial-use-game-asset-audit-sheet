# Free Game Font License Source Pack v0.1

This is a small verified starter pack for indie game developers who need practical font-license starting points.

## Contents

- `game_font_sources_verified_v0.1.csv`: 17 verified font/source rows.
- `font_license_questions_v0.1.csv`: 8 practical questions to ask before shipping a font in a game.

## Audit Method

Each verified row has:

- direct source URL
- evidence URL
- license signal
- commercial-use note
- attribution note
- redistribution note
- risk notes
- best-use note
- last checked date

For Google Fonts rows, the evidence URL points to an OFL license file in the official `google/fonts` repository. For GGBotNet rows, the evidence URL points to the source itch.io page checked on 2026-06-01.

## Limitations

This is not legal advice.

This pack does not redistribute font files. It is an audit/source directory and checklist. Before shipping, check the exact font file version you use, bundle the required license where needed, and separately review logo/trademark use.

## Why This Exists

Font licensing is a small problem until it blocks a release. This pack is meant to save the first round of research and give a repeatable audit format.
