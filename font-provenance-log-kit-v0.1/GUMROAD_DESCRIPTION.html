A free font provenance log kit for indie game developers.

Use this before shipping a game, jam build, demo, or prototype that embeds or distributes font files.

The kit is built for a common problem: a font came from a mirror, bundle, forum post, old prototype folder, or free-font site, but the original creator/source page and license evidence are unclear.

Included files:

  • font source-chain tracking CSV
  • mirror / reupload risk checklist CSV
  • before-shipping font recheck CSV
  • filled example provenance rows
  • workflow notes
  • audit notes and public-claim limits

It helps you track:

  • original source URL
  • evidence URL or license-file location
  • official source vs mirror/reupload risk
  • embedding and raw-file redistribution status
  • modification and reserved-name review
  • logo/title-use warning
  • final pre-release recheck status

Important: this is not legal advice. It does not include or redistribute font files or third-party game assets. It is a practical tracking kit for source URLs, evidence URLs, visible license signals, and before-shipping review.
