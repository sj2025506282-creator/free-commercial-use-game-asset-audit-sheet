# Font Provenance Log Kit v0.1

This is a free workflow kit for indie game developers who need to track where font files came from before shipping a game, jam build, demo, or prototype.

It is built for the common failure case where a font is found through a mirror, bundle, forum post, or "free font" site, but the original creator/source page and license evidence are unclear.

## Included Files

- `font_provenance_log_kit_v0.1.md`: main workflow notes.
- `font_provenance_log_template_v0.1.csv`: source-chain tracking table.
- `font_mirror_risk_checklist_v0.1.csv`: quick mirror/upload risk checks.
- `before_shipping_font_recheck_v0.1.csv`: final pre-release font review table.
- `filled_example_font_provenance_v0.1.csv`: small example rows showing how to use the fields.
- `AUDIT_NOTES.md`: scope, limits, and public-claim rules.

## What It Helps With

- Recording original source URL and evidence URL.
- Separating official foundry/platform sources from mirrors and reuploads.
- Tracking whether a font is embedded, modified, or redistributed with the game.
- Catching reserved font name, logo-use, and glyph coverage questions.
- Keeping a final re-check status before shipping.

## Important

This is not legal advice and does not replace a lawyer.
