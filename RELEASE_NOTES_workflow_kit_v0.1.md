# Game Jam Asset License Workflow Kit v0.1

This release packages the tracker as a practical workflow kit for game jams and prototype sprints.

## Included

- `game_jam_asset_license_workflow_kit_v0.1.xlsx`: workbook with Start Here, CC0 Starter, Attribution Tracker, Provenance Log, Font Checklist, and Before Shipping tabs.
- `game_jam_asset_license_workflow_kit_v0.1.zip`: all workflow kit files in one download.
- `cc0_starter_sources_v0.1.csv`: 42 CC0/no-attribution/no-signup starter source rows.
- `attribution_tracker_template_v0.1.csv`: template for CC BY or attribution-required assets.
- `provenance_log_template_v0.1.csv`: source/download/license evidence log template.
- `font_license_checklist_v0.1.csv`: font-specific checklist.
- `before_shipping_checklist_v0.1.csv`: final review checklist before publishing a game build.
- `preview_workflow_kit_v0.1.png`: preview image.
- `AUDIT_NOTES.md`: strict audit notes and public-claim limits.

## Safe Public Claim

A game-jam asset license workflow kit with CC0 starter sources, attribution tracking, provenance logging, font checks, and a before-shipping checklist.

## Important Limits

- This is not legal advice.
- It does not redistribute third-party asset files.
- It links to original creator/source pages.
- `creator_policy_signal` rows are weaker than direct page-level license signals and should be re-checked before shipping.
- Do not describe this as legally safe, lawyer-reviewed, guaranteed commercial use, all rights cleared, or an asset bundle.
