# Free Game Asset License Tracker v0.2

This release packages the tracker into a more usable format for indie game developers.

## Included

- `free_game_asset_license_tracker_v0.2.xlsx`
  - Summary
  - Asset License Tracker
  - Font License Tracker
  - Field Guide
- `game_asset_license_tracker_v0.2.csv`
- `font_license_tracker_v0.2.csv`
- `preview_license_tracker_v0.2.png`

## What Changed

- Added a Google Sheets/Excel-friendly workbook.
- Added a dedicated font licensing tracker.
- Added fields for embedding, modification, logo/brand use, editorial vs commercial use, and redistribution risk.
- Added a visual preview.

## Important

This is not legal advice.

The tracker links to original source pages and does not redistribute third-party asset files.

Before shipping a commercial game, verify the source page and license terms again.
