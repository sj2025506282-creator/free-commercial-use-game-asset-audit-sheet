## What This Is

This is a license tracking workbook for indie game developers who use free game assets.

It helps you track:

- license signals
- attribution requirements
- commercial-use notes
- redistribution risk
- source/provenance notes
- font licensing questions

## What This Is Not

This is not legal advice.

This package does not redistribute third-party asset files.

Before shipping a commercial game, verify each original source page and license terms again.

## How To Use

1. Open the XLSX in Excel or upload it to Google Sheets.
2. Use the filters on each tracker tab.
3. Start with the `CC0 Signals` tab if you want low-friction game-jam sources.
4. Keep source URLs and evidence URLs with your project notes.
5. Recheck licenses before shipping.
