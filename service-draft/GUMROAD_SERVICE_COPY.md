## Product Description

If you are building a game jam entry, prototype, or small indie game, you may already have a folder full of free asset links.

The problem is that "free" does not always mean the same thing:

- Some sources are CC0 or public domain.
- Some require attribution.
- Some have unclear commercial-use wording.
- Fonts may have separate embedding, modification, logo, or redistribution questions.
- Directory tags are weaker than license text on the creator or source page.

This starter audit is a small practical review pass.

Send up to 10 asset/source URLs. I will return a short report that flags:

- visible license signals
- CC0/no-attribution vs attribution-required cases
- commercial-use wording to re-check
- redistribution-risk notes
- font-specific questions if fonts are included
- before-shipping questions for your build
