# Game Jam Asset License Workflow Kit

This is a workflow version of the Free Game Asset License Tracker. It is built for indie developers who need to move from "I found free assets" to "I can track what I used and re-check it before shipping."

It does not include or redistribute third-party asset files.

![Preview](preview_workflow_kit_v0.1.png)

## What Is Included

- `game_jam_asset_license_workflow_kit_v0.1.xlsx`: workbook with Start Here, CC0 Starter, Attribution Tracker, Provenance Log, Font Checklist, and Before Shipping tabs.
- `cc0_starter_sources_v0.1.csv`: 42 CC0/no-attribution/no-signup starter rows.
- `attribution_tracker_template_v0.1.csv`: template for CC BY or attribution-required assets.
- `provenance_log_template_v0.1.csv`: source/download/license evidence log template.
- `font_license_checklist_v0.1.csv`: font-specific checklist based on the v0.2 font tracker.
- `before_shipping_checklist_v0.1.csv`: final review checklist before publishing a build.
- `AUDIT_NOTES.md`: public-claim limits and quality review.

## Use Case

Use this kit during a game jam or prototype sprint:

1. Pick likely sources from `CC0 Starter`.
2. Record anything you download in `Provenance Log`.
3. Put non-CC0 or attribution-required items in `Attribution Tracker`.
4. Check fonts separately in `Font Checklist`.
5. Run `Before Shipping` before publishing your game or public repo.
