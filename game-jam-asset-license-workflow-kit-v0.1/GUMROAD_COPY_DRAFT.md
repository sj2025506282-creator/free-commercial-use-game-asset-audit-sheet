# Gumroad Copy Draft

## Product Name

Game Jam Asset License Workflow Kit

## Short Description

CC0 starter sources, attribution tracker, provenance log, font license checklist, and before-shipping review template for indie game prototypes.

## Description

This is a small workflow kit for indie game developers who use free assets during game jams or prototype sprints.

It helps you:

- start from CC0/no-attribution/no-signup source rows
- record source URLs and license signals
- track attribution-required assets separately
- check font embedding/modification questions
- run a before-shipping checklist before publishing a build

Included:

- XLSX workbook with 5 workflow tabs
- CC0 starter source CSV
- attribution tracker template
- provenance log template
- font license checklist
- before-shipping checklist

Important:

This does not include or redistribute third-party asset files. It links to original source pages and provides workflow templates/risk notes. It is not legal advice.
