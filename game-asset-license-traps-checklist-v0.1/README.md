# Game Asset License Traps Checklist v0.1

This is a free checklist for indie developers and game-jam teams who use free game assets and want to catch obvious license/provenance problems before shipping.

It is built around seven common traps:

1. Treating "free" as one safe license bucket.
2. Mixing CC0 and attribution-required assets without tracking credits.
3. Assuming directory tags are stronger than the creator/source page.
4. Forgetting that fonts have their own licensing questions.
5. Redistributing raw asset files when the license only allows use in finished works.
6. Losing source URLs, download dates, or license evidence.
7. Releasing a commercial build without a final re-check.

## Included Files

- `game_asset_license_traps_checklist_v0.1.md`: the main checklist.
- `asset_license_red_flags_v0.1.csv`: quick red-flag review table.
- `font_license_questions_v0.1.csv`: font-specific questions for game projects.
- `source_provenance_log_v0.1.csv`: source URL and evidence log.
- `before_shipping_asset_review_v0.1.csv`: final before-shipping review table.

## How To Use

Use this before publishing a game jam build, prototype, demo, or commercial game.

The goal is not to prove that an asset is legally safe. The goal is to find obvious questions before they become release problems.

## Important

This is not legal advice and does not replace a lawyer.

This package does not include or redistribute third-party game assets. It only provides checklist and tracking files.
