# Starter Asset Audit Template Pack v0.1

This is a practical template pack for indie game developers, game jam teams, and prototype builders who need to organize free game asset sources before shipping.

It helps you track:

- visible license signals
- CC0 or public-domain candidates
- attribution-required assets
- source and provenance notes
- font-specific licensing questions
- before-shipping review items

## Included Files

- `starter_asset_audit_report_template.md`: short report template for reviewing asset/source URLs.
- `attribution_tracker_template_v0.1.csv`: table for attribution-required assets.
- `provenance_log_template_v0.1.csv`: table for recording source URLs, download notes, and license evidence.
- `font_license_checklist_v0.1.csv`: font-specific questions for games, apps, logos, and redistribution.
- `before_shipping_checklist_v0.1.csv`: final checklist before publishing a prototype or build.

## Suggested Use

1. Put every asset/source URL into the tracker before importing it into your game project.
2. Separate CC0/no-attribution assets from attribution-required assets.
3. Keep source URLs and license evidence, not just downloaded files.
4. Review font licenses separately from art/audio assets.
5. Re-check important sources before commercial release.

## Important

This pack is not legal advice and does not replace a lawyer.

It does not redistribute third-party game assets. It only provides templates for tracking source URLs, visible license signals, attribution notes, and before-shipping questions.
